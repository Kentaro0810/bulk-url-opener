# bulk-url-opener
# 一括URLオープンツール

複数のURLをまとめて新しいタブで開くWebアプリです。

## 機能

- 複数URLの入力
- 空行の自動除外
- `https://` の自動補完
- URLの簡易チェック
- 一括オープン
- URL一覧の保存
- URL一覧のクリア

## 使い方

1. テキストエリアにURLを1行ずつ入力する
2. 「URLを確認」を押して確認する
3. 「一括で開く」を押す
4. ブラウザで複数タブが開く

## 使用技術

- HTML
- CSS
- JavaScript
- GitHub Pages

## 注意事項

ブラウザのポップアップブロック設定によっては、複数タブの一括オープンが制限される場合があります。

その場合は、ブラウザ側でこのサイトのポップアップを許可してください。
